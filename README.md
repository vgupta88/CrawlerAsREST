# CrawlerAsREST


This project is a basic Spring boot application for crawling any url provided as per GET /api/{url}/crawl

## Requirement
  - JDK 1.8
  - Apache Maven 3.x

## Technologies
- Java 8
- Spring boot
- Jsoup

### Executing 
```sh
$ mvn spring-boot:run
```
